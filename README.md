# Proxima

Proxima is an open-source Astro application designed for small to medium stores, compatible with Shopify OS 2.0 themes. It renders Swellâ€™s flavor of Liquid templates (LiquidX) and can be used as a starter for custom storefronts or as a theme engine. For Swell hosting, it uses only Cloudflare Workers

## How to start

Switch your node version:

```bash
nvm install 18.16.1
nvm use
```

Install modules

```bash
npm i
```

Start in dev mode:

```bash
npm run dev
```
